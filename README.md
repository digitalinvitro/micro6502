# micro6502
6502 cpu based micro module (for mcu applications)

- 64K SRAM
- UART RX/TX
- load 256-byte MONITOR to SRAM from UART (terminal) on power on
- option spi, i2c, SN74HC595, dendy joystik interfaces programmed in CPLD
